# smart-hub

NOTE: Unfortunately, I did not realize this but free api keys from newsapi.org only support development testing builds, so the news component is not working for my live production build.

This is a personal hub dashboard built using React and Vite Framework. From just a simple glance the user can easily get the local date, local time, local weather, and top recent news stories. The user can also easily add tasks and keep track of what they have to do for the day, and easily delete them when they are completed.
